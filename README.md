# CRM Status Entscheidungsvorlage

Statische Website fuer GitHub und GitHub Pages auf Basis der CRM-Statusunterlage fuer Zwiesel Fortessa.

## Lokal ansehen

`index.html` direkt im Browser oeffnen.

## Auf GitHub veroeffentlichen

1. Ein neues Repository auf GitHub anlegen, zum Beispiel `crm-status-entscheidungsvorlage`.
2. Den Inhalt dieses Ordners in das Repository hochladen.
3. In GitHub unter `Settings` -> `Pages` die Quelle `Deploy from a branch` waehlen.
4. Branch `main` und Ordner `/ (root)` einstellen.
5. Danach ist die Seite unter der GitHub-Pages-URL verfuegbar.

## Projektstruktur

- `index.html`: komplette statische Website
- `README.md`: Kurzbeschreibung und GitHub-Pages-Hinweise
